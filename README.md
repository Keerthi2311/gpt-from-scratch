# gpt-from-scratch

Building a GPT from first principles, stage by stage, following
[Andrej Karpathy's Zero to Hero series](https://karpathy.ai/zero-to-hero.html).
Each numbered folder is a completed stage with its own README, code, and tests.

| Stage | Topic | Status |
|-------|-------|--------|
| [01-micrograd](./01-micrograd/) | Backprop from scratch — scalar autograd engine | ✅ Done |
| 02-makemore-bigram | Bigram language model, counting + NLL loss | 🔨 In progress |

More stages land as the series progresses.
